# Spotify Web Player Clone

Spotify Web Player Clone is a frontend UI project inspired by the Spotify web interface.
This project focuses purely on layout design and responsive styling using HTML and CSS.

Note: This is a static UI clone and does not include music playback functionality.

---

## Tech Stack

- HTML5
- CSS3
- Flexbox
- Media Queries

---

## Features

- Responsive sidebar navigation
- Playlist and album card layout
- Music player bottom bar UI
- Structured grid layout
- Responsive design for different screen sizes
- Clean and modern UI styling

---

## Project Objective

This project was built to strengthen:

- HTML structure and semantic elements
- CSS styling and positioning
- Flexbox layout techniques
- Responsive web design
- UI replication accuracy

---

## Installation & Usage

1. Clone the repository
   git clone https://github.com/rashi-v10/spotify-clone

2. Open the project folder

3. Open index.html in your browser

No additional setup required.
